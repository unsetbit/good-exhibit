About GOOD EXHIBIT

Hello, I'm Ozan Turgut, I created this website to express what I value. Specifically, my role in making this site is building an environment to hold the things I value and gluing the important pieces together. As such, any success of this website in actually exhibiting something good is primarily attributable to those whose works I rely on.

This website is an open-source project hosted on GitHub and served using CloudFlare's free CDN service.

Typography

True North is the display font used for the GOOD EXHIBIT title. Cindy Kinash created True North and dedicated it to her father, Matt Kinash who loved the outdoors.

Merriweather is the text font used through out GOOD EXHIBIT. Eben Sorkin created Merriweather, an open-source project which is continually being improved.
